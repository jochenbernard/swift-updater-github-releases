## Getting Started

Add the following dependency to your `Package.swift` file:

```Swift
.package(url: "https://github.com/jochenbernard/swift-updater-github-releases", from: "0.1.0")
```

## Minimum Versions

| Platform | Minimum Version |
| -------- | --------------- |
| macOS    | 12.0            |
